# 200B - Drinks

## Problem

There are `n` drinks, each containing a certain percentage of orange juice.

Find the percentage of orange juice in the final mixed drink.

---

## Approach

- Read the percentage of each drink.
- Add all percentages.
- Divide the total by the number of drinks.
- Print the average percentage.

---

## Algorithm

1. Read `n`.
2. Read all percentages and calculate their sum.
3. Compute `sum / n`.
4. Print the result.

---

## Complexity

- **Time Complexity:** `O(n)`
- **Space Complexity:** `O(1)`
