# 61A - Ultra-Fast Mathematician

## Problem

Given two binary strings of equal length, create a new binary string.

- If the bits are the same, output `0`.
- If the bits are different, output `1`.

---

## Approach

- Read both binary strings.
- Compare each pair of characters.
- Print `0` if they are equal.
- Otherwise, print `1`.

---

## Algorithm

1. Read the two strings.
2. Traverse both strings together.
3. Compare characters at each position.
4. Print the resulting binary string.

---

## Complexity

- **Time Complexity:** `O(n)`
- **Space Complexity:** `O(1)`
