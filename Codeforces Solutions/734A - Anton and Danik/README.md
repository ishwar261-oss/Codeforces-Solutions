# 734A - Anton and Danik

## Problem

Anton and Danik play `n` games.

- `A` → Anton wins
- `D` → Danik wins

Print:
- `Anton` if Anton wins more games.
- `Danik` if Danik wins more games.
- `Friendship` if both win the same number of games.

---

## Approach

- Count the number of `A` and `D`.
- Compare the counts.
- Print the winner.

---

## Algorithm

1. Read `n`.
2. Read the string.
3. Count `A` and `D`.
4. Compare the counts.
5. Print the result.

---

## Complexity

- Time: **O(n)**
- Space: **O(1)**
