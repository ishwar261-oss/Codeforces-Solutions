# 133A - HQ9+

## Problem

Given a program represented as a string.

If the string contains **H**, **Q**, or **9**, print **YES**.

Otherwise print **NO**.

---

## Example

### Input

```text
Codeforces
```

### Output

```text
NO
```

### Input

```text
Hi
```

### Output

```text
YES
```

---

## Approach

- Traverse the string.
- Check every character.
- If it is `H`, `Q`, or `9`, print `YES`.
- Otherwise print `NO`.

---

## Algorithm

1. Read the string.
2. Traverse every character.
3. If the character is `H`, `Q`, or `9`, print `YES`.
4. If no such character exists, print `NO`.

---

## Complexity

- Time: **O(n)**
- Space: **O(1)**
