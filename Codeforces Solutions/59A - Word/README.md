# 59A - Word

## Problem Statement

A word contains both uppercase and lowercase letters.

- If the number of uppercase letters is greater than lowercase letters, convert the entire word to uppercase.
- Otherwise, convert the entire word to lowercase.

Print the modified word.

---

## Approach

1. Count uppercase letters.
2. Count lowercase letters.
3. If uppercase letters are more, convert the whole word to uppercase.
4. Otherwise, convert the whole word to lowercase.

---

## Algorithm

1. Read the string.
2. Count uppercase and lowercase characters.
3. Compare their counts.
4. Convert the string accordingly.
5. Print the result.

---

## Complexity Analysis

- Time Complexity: O(n)
- Space Complexity: O(1)
