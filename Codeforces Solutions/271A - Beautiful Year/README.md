# 271A - Beautiful Year

## Problem

Given a year, find the next year in which all four digits are different.

---

## Example

Input

1987

Output

2013

---

## Approach

- Increase the year by 1.
- Check if all digits are unique.
- If yes, print the year.
- Otherwise, continue searching.

---

## Algorithm

1. Read the year.
2. Increment the year.
3. Extract each digit.
4. Check if any digit repeats.
5. If all digits are unique, print the year.

---

## Complexity

Time: O(1)

Space: O(1)
